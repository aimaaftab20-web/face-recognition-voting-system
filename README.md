# IoT-Based Electronic Voting System with Face Recognition

A secure electronic voting system that uses biometric face recognition for voter authentication, built using Raspberry Pi and computer vision technologies.

## 🚀 Features
- Biometric voter authentication using face recognition
- Real-time voter verification with OpenCV
- Secure database storage for tamper-proof records
- Automated and reliable voting process using IoT protocols

## 🛠 Technologies Used
- Python
- OpenCV
- Machine Learning
- Raspberry Pi
- IoT Systems

## 🎯 Applications
- Secure Digital Voting
- Biometric Authentication Systems
- Smart Governance Technologies
